The Basics · Object Creation & getName()

Create an object and supply three params (1) first name, (2) middle name, and (3) last name.

dude = new Name("Edward","Montgomery","Proundfoot");

You can then use the getName() method and supply a designated "type" as a param (see types below) to retrieve the name in your desired format.

str = dude.getName(type);

Types are a text string that uses single letters for initials (e.g., "f" for a first initial only), full words for spelled out words (e.g., "first-m-last" for first name and last name separated by an initial), and hyphens to designate spaces (e.g., "f-m-last" for last name preceded by initials but "fm-last" for same without space between initials).
